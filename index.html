Hello, Welcome to my website!

My name is Minh. As of 2027, I'm a first year PhD student at the RLPVLSI group at the University of Virginia. Advised by Prof. Calhoun, my current interests are high performance, robust and low-power circuits, architectures, and systems.

Some personal bio: I was orginally born in Hồ Chí Minh city, Việt Nam. I moved to California when I was 16. My English is not perfect, but I've gotten much better with time. Some of my hobbies includes pottery, climbing, guitar, photography, and learning Japanese (B1-ish level currently). On the technical end, I'm interested in enabling more efficient compute by means that are generally considered wacky, semiconductor materials & physics, and trading.

Before starting a PhD, I was advised by Prof. Balkind at UCSB for some of my undergrad, and my MS in ECE. My MS thesis is called "A RISC-V Implementation of The Sharing Architecture". Concisely, it was about enabling resource sharing between physical CPU cores to enable a more flexible organization depending on workload demand.

Some Projects

RISC-V Pipelined Core fully built in RTL and Cadence Custom Design FLow
This is a 5-stage core built in SystemVerilog. The core supports the base RV32I Extension set of the RISC-V ISA. The core is also fully built and tested in Cadence Virtuoso using the NCSU 45nm PDK. The clock speed is tested to be ~600Mhz based on the capacitive modelling at the time of the project.

Comparison between CMOS, CPL, CPL-TG for adders
A paper I wrote for the Computer Arithmetic course, where I explored the differences between using CMOS, CPL, CPL-TG for building low-power high-performance adders. Experiments were carried out in Synopsys HSPICE using the IBM130NM 8RF PDK, and gates were sized manually to ensure optimal VTC characteristics.
